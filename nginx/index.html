Welcome!

This is the default welcome page for the NGINX Home Assistant Add-on. If you can see this page, it means that the NGINX HTTP server has been successfully installed and configured.

Important: Before continuing to operate your HTTP server in production, either replace this file (/share/htdocs/index.html) with your own content, or configure a different document_root path in the add-on configuration to point to your website files.
