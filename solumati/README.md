## ⚙️ Configuration

Configure the app via the **Configuration** tab in the Home Assistant App page.

### Options

```yaml
app_base_url: ''
dev_use_main_branch: false
factory_reset: false
github_token: ''
log_level: info
marketing_page_enabled: false
test_mode: false
```
