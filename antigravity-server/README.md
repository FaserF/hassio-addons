## 📖 About

Stream the Antigravity AI IDE (Linux Desktop with XFCE4) via NoVNC in your browser.

---

## ⚙️ Configuration

Configure the app via the **Configuration** tab in the Home Assistant App page.

### Options

```yaml
autostart_antigravity: true
log_level: info
vnc_password: ''
```
