## Installation

1. Search for "ER-Startseite" in the Home Assistant App Store.
2. Click **Install**.
3. Start the app.
4. Click **Open Web UI**.

---

## ⚙️ Configuration

Configure the app via the **Configuration** tab in the Home Assistant App page.

### Options

```yaml
certfile: fullchain.pem
keyfile: privkey.pem
log_level: info
ssl: false
```
