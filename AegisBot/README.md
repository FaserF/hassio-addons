## âš™ï¸ Configuration

Configure the add-on via the **Configuration** tab in the Home Assistant App page.

### Options

```yaml
database:
  type: sqlite
debug: false
demo_mode: false
demo_mode_type: ephemeral
developer_mode: false
github_repo: FaserF/AegisBot
github_token: ''
log_level: info
project_name: AegisBot
reset_database: false
secret_key: ''
version: latest
```
