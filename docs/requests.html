Why I don't accept App requests

Thank you for your interest in my Home Assistant Apps repository! I often receive requests to add new apps, but I have decided not to accept them for the following reasons:

  • Private Use First: These apps are primarily developed and maintained for my own personal use. I only add software that I actually use in my own Home Assistant environment.
  • Support Quality: If I don't use an app myself, I cannot provide meaningful support or effectively debug issues. I want to ensure that every app in this repository meets a high standard of quality and reliability.
  • Maintenance Burden: Maintaining dozens of apps is a significant effort. Limiting the scope to apps I use helps me keep the repository healthy and up-to-date.

Note: Feature requests or bug reports for existing apps are always welcome! If you have an idea to improve one of the current apps, please feel free to open an issue.
