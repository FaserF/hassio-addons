# Home Assistant Add-on: BentoPDF

BentoPDF is a privacy-first, client-side PDF toolkit.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield] ![Supports armv7 Architecture][armv7-shield]

## About

BentoPDF is a comprehensive PDF toolkit that runs entirely in your browser. It allows you to merge, split, compress, and convert PDFs without uploading your files to a server, ensuring your data remains private.

## Installation

1.  Navigate to the **Add-on Store** in your Home Assistant supervisor.
2.  Search for "BentoPDF".
3.  Click **Install**.

## Configuration

**Note**: This add-on supports Ingress. You can access it directly from the Home Assistant sidebar.

### Ingress

The add-on is designed to run with Ingress.

## Support

If you encounter issues, please check the logs in the Add-on section of Home Assistant.
