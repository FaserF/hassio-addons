## Installation

1. Search for "Planka" in the Home Assistant App Store.
2. Install the app.
3. Start the app.

---

## ⚙️ Configuration

Configure the app via the **Configuration** tab in the Home Assistant App page.

### Options

```yaml
base_url: ''
certfile: fullchain.pem
keyfile: privkey.pem
log_level: info
reset_database: false
reset_database_confirm: false
secret_key: ''
ssl: false
```
