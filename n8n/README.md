## ⚙️ Configuration

Configure the app via the **Configuration** tab in the Home Assistant App page.

### Options

```yaml
certfile: fullchain.pem
keyfile: privkey.pem
listen_address: 0.0.0.0
log_level: info
port: 5678
ssl: false
```
