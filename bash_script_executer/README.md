## ⚙️ Configuration

Configure the app via the **Configuration** tab in the Home Assistant App page.

### Options

```yaml
log_level: info
script2_content: ''
script3_content: ''
script_content: ''
script_path: /share/scripts/mycoolscript.sh
script_path2: 'false'
script_path3: 'false'
```
